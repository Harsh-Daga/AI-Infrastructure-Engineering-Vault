# MCP

The Model Context Protocol (now under the Linux Foundation): the universal tool/data connectivity standard for agents. Transports (stateless/streamable HTTP), OAuth 2.1 auth, server discovery, and **progressive tool discovery** to fight context bloat — plus real security pitfalls.

> [!info] Why it matters for an infra engineer
> MCP is the production connectivity layer for agents. Operating servers (auth, discovery, security) is a fresh, shapeable infra skill.
